Edition window

To open the Soploon customization window, you have to press the edit toolbar button.


The customization window will be opened. This windows has two tabs.
  • The rules tab, which lists every rule already defined. On this tab you can and add new rules, and edit or desactivate existing ones.
  • The predicates tab, which lists predefined sets of auxiliary predicates included in Soploon to facilitate the teacherâ€™s work when specifying new rules. On this tab you can add new auxiliary predicates, and edit or delete existing ones.
