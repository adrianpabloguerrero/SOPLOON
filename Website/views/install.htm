Download

In order to install Soploon, first you have to download it from here.

Installation

Once you have downloaded Soploon.rar, you must follow the next instructions:
  1. First, you have to open the installation folder of Eclipse (where eclipse.exe is located).
  2. Once there, you have to open the folder named dropins.
  3. Inside dropins you have to create a folder named plugins.
  4. Finally, you have to drop the content of the Soploon.rar file into the folder plugins created in the previous step.
