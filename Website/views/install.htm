Install into Eclipse

Soploon is installable via the Eclipse software manager.

  1. Open the Eclipse Software Manager: Help â†’ Install New Software... (depending on your Eclipse version).


  2. Add the Soploon repository to the list of repositories: http://si.isistan.unicen.edu.ar/soploon/p2/


  3. Select Soploon from the list of features avaible in the repository.


  4. Click next and follow the instructions

Load Soploon configuration

You can configure Soploon to detect the errors you want. However, there is avaible a predefined set of rules and predicates that you can add to your Soploon.

The lastest version predicates and rules are avaible on the next links:

On the "Soploon Editor" window, you have the option to load external rules and auxiliary predicates from files.
