Soploon user interface

Soploons adds four components to the Eclipse UI.

  • The check toolbar button that starts the code analysis.
  • The edit toolbar button that opens the Soploon customization window.
  • The translation view that shows the Prolog representation of the last Java source code analyzed.
  • The corrections view that lists the errors detected in the last analysis.

How to analyze a Java project

  1. First, select a Java project in the Eclipse package explorer.


  2. Once you have selected the Java project, press the check toolbar button.


  3. When the analysis completes, the translation view will show you the Prolog representation of the selected Java project, and the corrections view will list the detected errors.


  4. Double-click any error and Soploon will take you to the code fragment containing the clicked error.
